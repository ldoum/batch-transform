# batch-transform
Sets new data for N objects of any type and N bones of selected armatures. 

## used the script to automate tasks to set up a police car model for a future video. 

## features
1. Apply axis lock
2. Set rotation mode 
3. Set new value for any axis property in euler or quaternion
4. Set coordinate values and applyy only if active
5. Press button to reset to default values
